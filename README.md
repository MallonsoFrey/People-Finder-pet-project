# 👤 User Management App

Простое веб-приложение для управления пользователями с возможностью просмотра, добавления, редактирования и удаления.

## 🚀 Функционал

- 📋 Просмотр списка пользователей
- 🔍 Поиск пользователей по имени
- 🔤 Сортировка (английский + русский алфавит)
- ➕ Добавление нового пользователя
- ✏️ Редактирование пользователя
- ❌ Удаление пользователя
- 📄 Детальная страница пользователя
- 📱 Ввод телефона с выбором страны и маской
- 💾 Сохранение данных через API
- 🔄 Обновление данных без перезагрузки страницы

---

## 🛠️ Технологии

- React
- TypeScript
- Redux Toolkit
- React Hook Form
- Axios
- React Router
- Material UI
- MockAPI (фейковый backend)
- react-phone-input-2

---

## 📦 Установка и запуск

```bash
git clone https://github.com/MallonsoFrey/People-Finder-pet-project.git
cd People-Finder-pet-project
npm install
npm run start
```

# 👤 User Management App

A simple web application for managing users with the ability to view, add, edit, and delete users.

## 🚀 Features

- 📋 View users list
- 🔍 Search users by name
- 🔤 Sorting (English + Russian alphabet support)
- ➕ Add new user
- ✏️ Edit user
- ❌ Delete user
- 📄 User details page
- 📱 Phone input with country selection and mask
- 💾 Data persistence via API
- 🔄 Real-time UI updates without page reload
